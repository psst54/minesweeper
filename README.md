## 📝 How to play

- 지뢰가 없는 칸에 좌클릭해서 칸을 열 수 있습니다
- 지뢰가 있는 칸에 우클릭해서 깃발을 세울 수 있습니다
- 숫자가 있는 칸을 클릭하면 주변 8칸을 동시에 열 수 있습니다
- 지뢰가 있는 모든 칸에 깃발을 세우면 승리합니다
- 남은 지뢰의 수는 보드 위에서 알 수 있습니다
